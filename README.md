🚀 EasyRoot VPS - Instant Root EnablerEasyRoot VPS adalah solusi satu baris perintah (one-liner) untuk mengaktifkan akses login root melalui SSH pada VPS Ubuntu Anda secara otomatis. Sangat berguna untuk VPS dari penyedia seperti AWS, Google Cloud, atau Azure yang secara default menonaktifkan akses root.✨ Fitur UnggulanOtomatisasi Total: Mengatur PermitRootLogin dan PasswordAuthentication dalam sekejap.Input Password Aman: Input password dibuat terlihat (visible) untuk meminimalisir risiko salah ketik yang berakibat gagal login.Validasi Ganda: Memastikan password dan konfirmasi cocok sebelum sistem melakukan perubahan.Ringkasan Informasi: Menampilkan detail login (Username & Password) dengan jeda 10 detik sebelum reboot agar Anda sempat mencatatnya.Smart Reboot: Melakukan pembersihan sesi dan reboot otomatis untuk menerapkan konfigurasi baru.🖥️ Cara PenggunaanCukup salin dan tempel perintah di bawah ini ke terminal VPS Anda (pastikan Anda login sebagai user biasa yang memiliki akses sudo):bash <(curl -Ls [https://raw.githubusercontent.com/syamsul18782/ngeroot/main/easyroot.sh](https://raw.githubusercontent.com/syamsul18782/ngeroot/main/easyroot.sh))
🛠️ Alur Kerja SkripInisialisasi: Membersihkan layar dan menampilkan peringatan input password.Setup Password: Menghasilkan password root sesuai input manual Anda.Konfigurasi SSH:Mencari dan mengubah baris konfigurasi di /etc/ssh/sshd_config.Mengaktifkan akses password dan login root secara eksplisit.Verifikasi: Menampilkan teks konfirmasi detail akun root Anda.Finalisasi: Menghitung mundur 10 detik, kemudian melakukan reboot sistem.⚠️ Peringatan KeamananMengaktifkan akses root dengan autentikasi password meningkatkan risiko terhadap serangan brute force.Gunakanlah password yang sangat kuat (gabungan huruf besar, kecil, angka, dan simbol).Disarankan untuk segera mengganti password secara berkala atau menggunakan SSH Key setelah akses root berhasil diaktifkan.👤 KontributorSyamsul - syamsul18782📝 CatatanSkrip ini didesain khusus untuk keluarga sistem operasi Debian/Ubuntu. Penggunaan pada distro lain mungkin memerlukan penyesuaian pada lokasi file konfigurasi SSH.
